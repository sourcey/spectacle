Cheese Store API Reference

Welcome to the sample Cheese Store API reference. This is a live example of how you can use Spectacle in conjunction with Swagger to generate beautiful static documentation for your own APIs.

The Cheese Store API is organized around the REST methodology, and it uses resource-oriented URLs, and common HTTP response codes to indicate API errors. All requests are authenticated using an api-key which can be obtained from your developer dashboard. And now, more cheese...
